# 💰 SpendSmart – Personal Expense Manager

A modern, premium-quality expense manager built using  
**HTML + CSS + Vanilla JavaScript**, designed like a real product used in companies.

---

## 🚀 Features

### ✔ Add expenses  
- Description, Amount, Date  
- Validates input and prevents negative values

### ✔ Delete entries  
- Remove individual expenses with one click

### ✔ Filters  
- Date range filter
- Clear filters to view all records

### ✔ Summary  
- Live total calculation displayed underneath filters

### ✔ Persistent storage  
- Data saved in localStorage so entries survive refreshes

### ✔ Premium UI  
- Animated night-sky heading, snow effect background
- Responsive layout and hover animations
- Light/Dark theme toggle 🔁

### ✔ Cross-device friendly  
- Works on mobiles and desktops
